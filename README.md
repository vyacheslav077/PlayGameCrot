# PlayGameCrot: Игра "Крот-охотник"

## Название: Какую проблему решает проект
Проект решает проблему отсутствия простых развлекательных игр для тренировки внимания и реакции. Это мини-игра, где игрок угадывает позицию "крота" на поле, чтобы "убить" его, проходя уровни с ограниченным количеством жизней.

## Целевая аудитория
- Студенты и школьники, интересующиеся программированием и играми.
- Любители простых пазлов и аркад (возраст 10+).
- Пользователи, желающие потренировать логику и координацию.

## Цели и задачи
- **Цель:** Разработать десктопное приложение с GUI, демонстрирующее навыки алгоритмизации, интерфейса и логирования.
- **Задачи:**
  - Создать игру с уровнями сложности.
  - Реализовать GUI с кнопками и полями ввода.
  - Автоматически логировать действия в файл.
  - Сохранять результаты игр.
  - Документировать по методичке.

## Функционал
- Главное меню: Кнопки "Старт", "Выход".
- Игровое окно: Поле с кнопками (клик для "удара"), отображение жизней, уровней.
- Проверка: Программа проверяет угадывание, уменьшает жизни, переходит на уровни.
- Сохранение: Результаты (победа/поражение, дата) в `results.txt`.
- Логирование: Автоматическое в `game_log.txt` (каждый ход, с timestamp).
- Несколько окон: Меню -> Игра -> Результаты.
